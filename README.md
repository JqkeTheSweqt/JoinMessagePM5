# JoinMessage
Sends a message to your players upon joining!


# How To Use
To upload this plugin to your server follow these steps:
1. Ensure you have DevTools installed.
2. Click Code -> Download Zip
3. Extract the contents of the zip
4. Upload to your servers plugin folder via SFTP

# How to customize the join message
To customize the join message follow these steps:
1. Once you have uploaded the plugin to your server, start & stop the server.
2. Find & Open the plugin_data folder in your server files
3. Open "JoinMessage" & open the config.yml
4. Edit the join message in the config.yml to be whatever you want your join message to be!
